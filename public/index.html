Posts

Reading

Send a request to read all posts using a GET request:

fetch('http://localhost:3000/post').then(data => processData(data)).catch(err => handleError(err));

Reading by id

Send a request to read a post by its id using a GET request, include it as a path variable:

fetch('http://localhost:3000/post/624f543911135c2c00a2c7fa').then(data => processData(data)).catch(err => handleError(err));
